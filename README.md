# LUTOhayse.bots
Система создания и отслеживания заказов для фабрики мебели LUTOhayse. Через чат-боты пользователи могут создавать свои диваны и заказывать их вместе с готовыми продуктами, а также отслеживать их прогресс в доставке. Работники могут отслеживать порученные им задачи и менять их статус.
